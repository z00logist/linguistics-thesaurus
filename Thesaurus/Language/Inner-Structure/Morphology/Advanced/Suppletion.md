# Suppletion

Suppletion is a morphological phenomenon where an irregular morphological change occurs, and a morpheme is replaced entirely by a different morpheme.


## Examples

- **English**:
  - "Go" → "Went" (past tense)
  - "Good" → "Better" → "Best" (comparative and superlative)
- **Spanish**:
  - "Ir" (to go) → "Fui" (I went)
